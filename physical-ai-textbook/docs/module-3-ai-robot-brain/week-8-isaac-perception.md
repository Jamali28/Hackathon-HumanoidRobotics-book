# Module 3: AI-Robot Brain - Week 8: Introduction to NVIDIA Isaac and Perception

This week introduces the NVIDIA Isaac platform, focusing on Isaac Sim for simulation and Isaac ROS for GPU-accelerated perception pipelines, including VSLAM.

## Learning Objectives

*   Understand the NVIDIA Isaac ecosystem.
*   Explore Isaac Sim for robot simulation.
*   Implement basic perception tasks using Isaac ROS.

## Lab Exercises

*   **Lab 8.1**: Launch an Isaac Sim environment and interact with a robot.
*   **Lab 8.2**: Implement a basic VSLAM pipeline using Isaac ROS.
