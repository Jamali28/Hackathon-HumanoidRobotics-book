# Module 3: AI-Robot Brain - Week 10: Navigation and Path Planning

This week focuses on robot navigation, covering various path planning algorithms and their implementation within the Isaac ecosystem.

## Learning Objectives

*   Understand global and local path planning.
*   Implement obstacle avoidance strategies.
*   Integrate navigation with Isaac Sim environments.

## Lab Exercises

*   **Lab 10.1**: Implement a global path planner for a given map.
*   **Lab 10.2**: Navigate a robot through an obstacle course in Isaac Sim.
