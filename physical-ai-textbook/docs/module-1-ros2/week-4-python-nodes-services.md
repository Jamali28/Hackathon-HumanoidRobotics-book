# Module 1: ROS 2 - Week 4: Python Nodes and Services

This week, we dive deeper into developing ROS 2 applications using Python, focusing on creating custom nodes and implementing services.

## Learning Objectives

*   Develop ROS 2 nodes in Python.
*   Implement ROS 2 services for request/response communication.
*   Understand best practices for Python in ROS 2.

## Lab Exercises

*   **Lab 4.1**: Create a custom Python ROS 2 node.
*   **Lab 4.2**: Implement a simple calculator service.
