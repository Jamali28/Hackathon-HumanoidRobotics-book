# Module 2: ROS 2 Basics

This module covers the fundamentals of Robot Operating System 2 (ROS 2), an open-source framework for developing robotic applications.

## Topics Covered:
- ROS 2 architecture and concepts (nodes, topics, services, actions).
- Setting up a ROS 2 workspace.
- Basic command-line tools.
- Writing simple ROS 2 nodes in Python.
